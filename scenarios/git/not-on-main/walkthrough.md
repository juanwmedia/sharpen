Create the branch pointer first, move HEAD onto it, then stage and commit the untracked banner. Canonical path: `git branch feature/welcome-banner`, `git switch feature/welcome-banner`, `git add src/banner.ts`, then `git commit -m`. `git switch -c feature/welcome-banner` (or `git checkout -b`) is the same idea in one step and also passes. Do not commit on `main` and move the tip later unless you leave `main` at the original commit. Any state-correct route passes.
